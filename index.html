Yet Another Minimalistic Template Engine

YAMTE is obviously a minimalistic template engine build with PHP 5. You can either use the PHP 5.2 or the PHP 5.3 version. It has been built as a fork of Vtemplate, but only the syntax of template files has been kept. For more information, have a look at the README.md

Dependencies

PHP 5 (5.3 if you want to use the namespaces, 5.2 either)

Install

Just download the lib and require_once it in your PHP file !
